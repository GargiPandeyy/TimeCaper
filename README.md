# Time-Caper Chronicle

A time-travel educational game where you play as an agent of the Time Variance Society, hunting down anachronisms across different historical eras.

## How to Play

1. Select a mission from the Time Hub
2. Use your scanner to find the anachronism
3. Answer quiz questions to secure the artifact
4. Collect trophies in your Trophy Room

## Features

- Interactive scanner with distance detection
- Historical quiz questions for each era
- Trophy collection system
- Immersive terminal-style interface
- Audio effects and visual animations

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Web Audio API
- Local Storage

I made this with a lot of struggle but loved doing it and learning all the things while building it.
